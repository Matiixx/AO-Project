# AO-Project

Rozpoznawanie znaków z tablicy rejestracyjnej ze zdjęcia samochodu

## Instalacja i uruchomienie

Instalacja odpowiednich bibliotek

```bash
  pip install -r requirements.txt
```

Uruchomienie

```bash
  python3 main.py
```
